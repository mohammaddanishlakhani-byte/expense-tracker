# ₹ Expense Tracker

A simple, beautiful expense tracking web application built with HTML, CSS, and JavaScript.

## Features

- ✅ Add income and expense transactions
- ✅ Categorize transactions (Food, Transportation, Entertainment, etc.)
- ✅ Real-time financial summary (Income, Expenses, Balance)
- ✅ Indian Rupees (₹) support with proper formatting
- ✅ Dark/Light mode toggle
- ✅ Data persistence with localStorage
- ✅ Export/Import functionality
- ✅ Responsive design for all devices
- ✅ Beautiful charts for expense analysis
- ✅ Search and filter transactions

## How to Use

1. **Open `index.html`** in any modern web browser
2. **Add a transaction**:
   - Select Expense or Income tab
   - Enter amount in ₹
   - Add description
   - Choose category (click on icons)
   - Select payment method
   - Click "Add Transaction"

3. **View your finances**:
   - Check overview cards for totals
   - See pie chart for expense distribution
   - Browse transaction history

4. **Manage data**:
   - **Export**: Click Export button to backup data
   - **Import**: Paste JSON data to restore
   - **Clear**: Remove all transactions with Clear All
